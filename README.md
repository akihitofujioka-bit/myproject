# 議会だより作成支援ソフト

議員から提出された記事（Word・Excel等）を、決められたレイアウトに流し込んで
議会だより（広報紙）を作成するためのデスクトップアプリです。

## 主な機能（計画）

- 提出記事の取り込み・正規化（Word / Excel / PDF / 手書き原稿のスキャン）
  - 様式バラバラの提出物を、アプリ内で同じ記事データに整える「正規化ワークベンチ」
  - 手書き原稿はスキャン画像を見ながら手入力で書き起こし（オフライン完結）
- 記事本文の編集（縦書き対応）
- 写真・画像の簡単な編集（切り抜き・回転・明るさ調整）
- 議員の掲載順の決定（ドラッグ＆ドロップ並べ替え）
- レイアウトテンプレートへの自動流し込み
- Word / PDF での出力

## 形態

- Electron によるデスクトップアプリ（Windows / Mac）
- オフライン・ローカル完結（原稿や議員情報を外部送信しない）

## 開発

```bash
npm install
npm run dev        # 開発起動
npm run build      # ビルド
npm run test:core  # 中核ロジックの単体テスト
```

詳細は [開発セットアップ](docs/dev-setup.md) を参照。
